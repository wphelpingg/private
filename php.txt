Install PHP on Ubuntu

https://www.digitalocean.com/community/tutorials/how-to-install-php-7-4-and-set-up-a-local-development-environment-on-ubuntu-18-04

STEP 1. sudo add-apt-repository ppa:ondrej/php
STEP 2. sudo apt-get update
		sudo apt-get install php7.4
		sudo a2enmod php7.4
		sudo a2dismod php7.3
		sudo systemctl restart apache2

		sudo apt-get install php7.4-common php7.4-mysql php7.4-mbstring libapache2-mod-php7.4
		sudo systemctl restart apache2

STEP 3. sudo apt-get -y install php8.3-common php8.3-mysql libapache2-mod-php8.3 php8.3-mcrypt php8.3-mbstring php8.3-curl php8.3-cli php8.3-mysql php8.3-gd php8.3-intl php8.3-xsl php8.3-zip
